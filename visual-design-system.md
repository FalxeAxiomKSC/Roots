# Visual Design System: Solarpunk Aesthetic for Sustainability Funding

## Overview
This design system establishes a cohesive visual language for the sustainability funding surfacing interface, rooted in solarpunk principles of harmony between technology and nature, community empowerment, and sustainable development.

---

## 1. Design Philosophy

### Solarpunk Principles
- **Harmony with Nature**: Technology serves ecological and social well-being
- **Community-Centered**: Design facilitates collaboration and shared prosperity
- **Sustainable Innovation**: Solutions that regenerate rather than extract
- **Democratic Access**: Tools and resources available to all, not just the privileged
- **Slow Technology**: Thoughtful, deliberate design that serves real needs

### Visual Expression
- **Organic Forms**: Natural shapes and flowing lines
- **Biophilic Design**: Connection to nature through color, pattern, and imagery
- **Warm & Welcoming**: Accessible design that builds trust
- **Clear & Purposeful**: Every element has meaning and function
- **Community-Focused**: Visual language that emphasizes collaboration

## 3. Typography

### Font Selection
- **Primary**: Inter (Modern, highly readable, excellent screen rendering)
- **Secondary**: Lato (Friendly, approachable for body text)
- **Headlines**: Playfair Display (Elegant, sophisticated for titles)

### Typography Scale
```
H1: 48px / 56px    - Page titles and main headings
H2: 36px / 42px    - Section headings
H3: 28px / 32px    - Card titles and important UI elements
H4: 22px / 26px    - Subheadings and labels
Body: 16px / 24px  - Main content
Small: 14px / 20px  - Secondary text and captions
```

### Typography Hierarchy
```
🎯 Primary Text: #263238 (Dark, high contrast)
📝 Secondary Text: #546E7A (Medium contrast)
🏷️ Labels: #78909C (Low contrast, supporting)
🔘 Disabled: #B0BEC5 (Very low contrast)
```

### Usage Guidelines
- **Headings**: Playfair Display, bold, for main page titles
- **Subheadings**: Inter, semibold, for section titles
- **Body Text**: Inter, regular for main content
- **Labels & Small Text**: Lato, regular for supporting text

### Grid System
- **12-column grid** for consistency and flexibility

### Spacing Scale
```
Micro: 4px   - Between UI elements
Small: 8px   - Component padding

### Layout Principles
- **Breathing Room**: Ample whitespace for visual clarity

- **Responsive Design**: Mobile-first approach with adaptive layouts

## 8. Animation & Interaction

### Motion Principles
- **Purposeful**: Every animation has a clear purpose
- **Smooth**: Natural, easing transitions (ease-in-out)
- **Fast**: Quick response times (150-300ms)
- **Accessible**: Respect user preferences for motion

### Animation Types
```
Micro-interactions
├── Button hover: Scale 1.05, color transition
├── Card hover: Lift 4px, shadow increase
├── Form focus: Border color transition
├── Loading spinner: Gentle rotation
└── Success checkmark: Check animation

Page Transitions
├── Fade in/out: 300ms duration
├── Slide from right: Navigation changes
├── Scale in: Modal/dialog appearance
└── Stagger reveal: List items loading

Data Animations
├── Progress bars: Smooth fill animation
├── Chart updates: Gentle data transitions
├── Number counters: Smooth counting up
└── Status changes: Gentle pulse effect
```

### Interaction States
```
Default: Normal state
├── Background: As designed
├── Border: As designed
├── Shadow: As designed
└── Cursor: Pointer (if interactive)

Hover: Mouse over element
├── Background: Lightened/darkened
├── Border: Slightly thicker
├── Shadow: Increased depth
└── Cursor: Pointer

Active: Being clicked/touched
├── Background: Further changed
├── Border: Thicker
├── Shadow: Reduced (pressed effect)
└── Cursor: Pointer

Focus: Keyboard navigation
├── Border: Highlighted with focus ring
├── Background: Slightly changed
├── Shadow: Focus indicator
└── Outline: 2px focus ring

Disabled: Not interactive
├── Background: #F5F5F5
├── Text: #B0BEC5
├── Border: #E0E0E0
├── Cursor: Not-allowed
└── Shadow: None
```

## 9. Accessibility Standards

### Color Contrast
- **Minimum AA**: 4.5:1 for normal text
- **Minimum AAA**: 7:1 for large text
- **Interactive elements**: 3:1 contrast
- **All combinations** meet WCAG standards

### Text Accessibility
- **Minimum size**: 16px for body text
- **Line height**: 1.5x for readability
- **Font choice**: Highly readable sans-serif
- **Responsive scaling**: User-controllable text size

### Keyboard Navigation
- **Full keyboard access** for all interactive elements
- **Logical tab order** through interface
- **Focus indicators** clearly visible
- **Skip navigation** options for screen readers

### Screen Reader Support
- **Semantic HTML** for proper structure
- **ARIA labels** for complex components
- **Alt text** for all meaningful images
- **Proper heading hierarchy** (H1-H6)
